## Learning Progression

```
Std 1–2  →  Build circuits with hands. No code. Pure electricity and motion.
Std 3–4  →  Advanced circuits. RC car with buttons. Electromagnets. Hovercraft.
Std 5–6  →  First Arduino. Every major sensor. Health monitors. Earthquake detectors.
Std 6–7  →  Logic gates. Transistors. 555 timer. Circuits that work with zero microcontroller.
Std 7–8  →  Full robots. FlySky RC. Bluetooth. Stepper motors. Autonomous systems.
Std 8–9  →  IoT cloud. AI models. Real-world capstone. Genuinely impressive prototypes.
```

A student who completes all 5 levels has:
- Built real machines from scratch with no computer (Level 1)
- Programmed every major Arduino sensor combination (Level 2)
- Understood electronics at the IC/transistor level (Level 3)
- Built autonomous RC robots with multiple wireless protocols (Level 4)
- Connected hardware to the internet and trained AI models (Level 5)

## Component Master List (All Levels)

### Electrical Basics (Level 1)
- LEDs (multi-color), 220Ω resistors, coin cells, AA/9V batteries, battery holders
- Push buttons, slide switches, toggle switches
- Active buzzer, DC motors (3–6V), propeller blades
- Copper tape, conductive thread, breadboards, jumper wires, alligator clips
- L298N motor driver (for button-controlled RC car)

### Arduino Ecosystem (Levels 2–4)
- Arduino Uno (Level 2), Arduino Nano (Levels 3–4)
- Breadboard, jumper wires, USB cables (A-B for Uno, Mini for Nano)

### Sensors (Levels 2–5)
- LDR, IR sensor module, PIR motion sensor
- DHT22 (temperature + humidity)
- HC-SR04 ultrasonic sensor × 2
- Soil moisture sensor
- MPU-6050 gyroscope/accelerometer
- Pulse sensor (heart rate)
- MQ-2 gas/smoke sensor, MQ-135 air quality sensor
- TTP223 capacitive touch sensor
- RC522 RFID module + cards

### Actuators
- Servo SG90 × 3, Stepper 28BYJ-48 + ULN2003 driver
- L298N motor driver, DC motors × 2
- 5V relay module

### Display
- OLED 0.96" I2C, 16×2 LCD I2C
- 7-segment display (single digit), 8-LED bar graph

### Wireless + Communication (Levels 4–5)
- HC-05 Bluetooth module
- NodeMCU ESP8266 / ESP32
- FlySky FS-i6 transmitter + FS-iA6B receiver
- TSOP1738 IR receiver
- HC-12 radio module (optional)
- SIM800L GSM module (optional)

### Electronics Components (Level 3)
- BC547 NPN transistor, BC557 PNP transistor
- ICs: 7408 (AND), 7432 (OR), 7404 (NOT), 7486 (XOR), 7400 (NAND)
- NE555 timer IC, LM358 op-amp IC, 7805 voltage regulator
- CD4026 counter IC, L293D motor driver IC
- Resistors: 220Ω, 1kΩ, 10kΩ
- Capacitors: 10µF, 100µF, 0.01µF ceramic

## What Makes This Curriculum Different

1. **Level 1 has zero computers** — students discover that machines can do impressive things with just wires, motors, and batteries. This builds genuine understanding before abstraction.
2. **Level 2 uses every major sensor** — no cherry-picking easy ones. Students use MPU-6050, pulse sensor, MQ-2, and more in their first Arduino year.
3. **Level 3 teaches the silicon** — most curricula skip transistors and ICs entirely. Students here build circuits that think without microcontrollers.
4. **Level 4 uses professional RC hardware** — FlySky FS-i6 is used by real hobbyist RC pilots. Students use the same equipment.
5. **Level 5 connects to the real world** — IoT projects serve live dashboards. AI projects train real ML models. Year-end projects look like startup prototypes.
6. **Every level has a mid-year checkpoint** — validates learning before the year-end sprint.
7. **Projects never repeat across levels** — each year brings genuinely new challenges and hardware.
