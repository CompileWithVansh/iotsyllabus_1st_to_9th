## 🔗 The Hierarchy Chain Project — 9-Year Upgrade Table

The same robot evolves across all 9 years. Every student keeps their project and upgrades it the next year.

| Year | Chain Project | Technology Added | WOW Moment |
|------|--------------|-----------------|------------|
| **1st** | Vibrating Bug | DC motor + asymmetric weight + decorated body | It CRAWLS across the table with no code! |
| **2nd** | Bug → RC Car | L298N motor driver + 4 push buttons + chassis | Drive it F/B/L/R with physical buttons! |
| **3rd** | RC Car + Headlights + Horn | Parallel LEDs + buzzer + capacitor-timed horn | Dim the room — real headlights glow. Press horn → 2-second honk! |
| **4th** | Auto-Headlights + 555 Blink Light | BC547 transistor + LDR + NE555 timer | Dark → lights come on automatically. No switch needed! |
| **5th** | Arduino Sensor Car | Arduino Uno + IR sensor + LDR code | Car detects obstacle → stops itself before crashing! |
| **6th** | LCD Dashboard Car | HC-SR04 ultrasonic + 16×2 LCD | LCD counts down distance as car approaches wall. Speed zones! |
| **7th** | Autonomous Scanning Robot | Arduino Nano + servo-scanner + OLED | Robot SCANS left/right, DECIDES which way to turn. Genuinely autonomous! |
| **8th** | Dual-Mode IoT Robot | HC-05 Bluetooth + ESP8266 + Blynk | Manual phone control OR full autonomy. Live dashboard on Blynk! |
| **9th** | AI Gesture Robot | ESP32 + Teachable Machine + Python | ✋👆👈👉✌️ — hand gestures control the robot. No buttons. No app. Pure AI! |

## 📚 Two Projects Per Year

Every year has exactly two projects:

### 🔗 Chain Project
The flagship robot that grows and evolves over 9 years. Students keep it, upgrade it, and present it every year. The 9th-class AI gesture robot contains physical DNA from the 1st-class vibrating bug!

### 🚀 Standalone Problem-Solving Project
A completely separate project that addresses a real-world problem. New every year, never repeated.

| Year | Standalone Project | Real-World Problem Addressed |
|------|-------------------|------------------------------|
| 1st | Mini Fire Alarm | Early fire detection and warning |
| 2nd | Traffic Light System | Intersection signal control |
| 3rd | Burglar Alarm System | Intruder detection + alert |
| 4th | Logic Gate Combination Lock | Secure access control |
| 5th | Smart Health Monitor | Home vitals monitoring (BPM + temperature) |
| 6th | Gas Leak & Fire Detector | Kitchen safety — dual threat detection |
| 7th | Smart Parking System | Real-time parking slot availability |
| 8th | Smart Home Control Panel | Remote device control + auto-alerts |
| 9th | Capstone (student choice) | AI waste sorting / smart agriculture / adaptive traffic / energy monitoring |

## 🎓 Learning Progression

```
Std 1  →  Battery + LED + Motor. No code. Electricity is real and exciting.
Std 2  →  Motor driver + 4 buttons = direction control. Engineering with your hands.
Std 3  →  Diodes, capacitors, LEDs on a moving car. Features make things better.
Std 4  →  Transistors, logic gates, 555 timer. Circuits that think without code.
Std 5  →  Arduino Uno. Code enters the lab. Sensors respond. The car thinks!
Std 6  →  Every sensor has a story. Distance. Gas. Tilt. The world is measurable.
Std 7  →  Robots that scan and decide. True autonomous behavior.
Std 8  →  Wireless everything. Phone control. IoT dashboards. Connected hardware.
Std 9  →  AI meets hardware. Train a model. Control the world with your hands.
```

A student who completes all 9 years can:
- Build real machines without a computer (Years 1–4)
- Program and debug Arduino projects from scratch (Years 5–7)
- Connect sensors to the internet and build dashboards (Year 8)
- Train AI models and connect them to physical hardware (Year 9)
- Present an engineering solution: problem → design → build → test → demo

## 🏆 Year-End Exhibition

Every year ends with a public exhibition. Visitors, parents, and teachers attend. Students demonstrate BOTH projects:
- The Chain Project shows HOW FAR THE ROBOT HAS COME since 1st class
- The Standalone Project shows HOW THEY SOLVE REAL PROBLEMS

The 9th-class exhibition is the finale: side-by-side display of a 1st-class vibrating bug and a 9th-class AI gesture robot. Same DNA. 9 years of upgrades.
